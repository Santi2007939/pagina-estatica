Código Binario

Es un sistema numérico en base 2, es un sistema de codificación de texto o procesadores de instrucciones de computadora, este sistema numérico solo cuenta con las cifras 0 y 1. En informática y telecomunicaciones es utilizado para la codificación de datos, como cadenas de caracteres o bits.

Cifrado César - ROT 13

Es un cifrado César utilizado para ocultar un texto reemplazando letra por letra trece posiciones por delante. Se utiliza la misma clave para cifrar y descifrar, por ello no es necesario intercambiar las posiciones para obtener un resultado diferente.

Reversa

Como dice el titulo, solo escribe el texto que tú desees y se colocara en reversa.
